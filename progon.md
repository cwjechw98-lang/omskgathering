Перед внесением изменений в игровой движок необходимо проверить, что механики игры работают корректно.

Выполнить следующие шаги:

1. Запустить unit тесты проекта.

npm run test

2. Запустить regression тесты механик боя.

npm run test:regression

3. Если добавлена система replay, запустить replay regression тест.

npm run test:replay

4. Проверить, что все тесты проходят без ошибок.

Если тест падает:

* определить причину
* исправить проблему
* повторно запустить тесты

Не вносить изменения, которые ломают существующие regression тесты.

После успешного прогона:

* зафиксировать изменения через существующую систему логирования проекта
* выполнить commit
* выполнить push
