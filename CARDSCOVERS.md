# CARDSCOVERS.md

Подробные промпты для генерации артов в стилистике **Magic: The Gathering** по всему обновлённому кодексу `CARDS_CODEX-kopiia.md` с учётом лора мира, фракций Омска, мемных архетипов, зимней атмосферы, метро-мифологии и цифровой магии Школы 21.

## Как использовать

- Каждый блок ниже подписан названием карты.
- Все промпты рассчитаны на квадратный формат **1:1**.
- Арт должен выглядеть как премиальная иллюстрация для TCG: серьёзно, атмосферно, без пародийной дешёвизны.
- Для мемных карт важна не просто шутка, а узнаваемый локальный миф, превращённый в эпическое фэнтези.
- Текст на самом изображении не нужен: только иллюстрация.

## Общий технический хвост

`Magic: The Gathering style fantasy card illustration, premium trading card art, square 1:1, highly detailed, painterly rendering, cinematic lighting, strong silhouette, layered atmosphere, clean focal composition, no text, no card frame, no UI`
